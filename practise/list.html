Block Elements

Block elements are those which start from a new line.They take complete width of the screen.

Example:
  1. h1 to h6
  2. p
  3. ul
  4. ol

Inline Elements

Inline elements which start from the point where the previous element ended.

Example:
  • a
  • img
  • button
  • select
